# Descrição do Modelo Hidrológico

Este documento apresenta a estrutura inicial do modelo hidrológico da bacia do Murucutum.

## Tipo de modelo
Modelagem hidrológica utilizando o software SWMM.

## Componentes previstos
- Sub-bacias
- Nós de drenagem
- Condutos
- Dados de precipitação

## Status
Em fase inicial de estruturação
