# swmm-murucutum-model
# Modelagem Hidrológico-Hidráulica da Sub-bacia do Murucututu

## Descrição
Este repositório apresenta o desenvolvimento de um modelo hidrológico-hidráulico utilizando o software SWMM aplicado à bacia do Murucutum, localizada em Belém-PA.

## Objetivo
Simular o comportamento do escoamento superficial em eventos de precipitação intensa, com foco na identificação de pontos críticos de alagamento e na avaliação de soluções baseadas na natureza.

## Metodologia
- Modelagem no SWMM
- Dados de precipitação e uso do solo
- Simulação de cenários atuais e com intervenções

## Aplicação
Planejamento de drenagem urbana sustentável em áreas críticas de Belém-PA.
